# Version Control System & Versioning Management


## Version Control System
This project uses Git for version control.  

Please see [git-scm.com](https://git-scm.com/) for more details. 

## Versioning 

This project uses Semantic Versioning 2.0.0

For further information please visit [semver.org](https://semver.org/)


### Summary of Versioning System Used

Given a version number MAJOR.MINOR.PATCH, increment the:

 * MAJOR version when you make incompatible API changes,
 * MINOR version when you add functionality in a backwards compatible manner, and
 * PATCH version when you make backwards compatible bug fixes.
